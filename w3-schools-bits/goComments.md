## Go Comments

Commenting in Go:
* Is used to stop a particular block of code from execution when testing another code.
* Is used to make a code readable.
* It is also used to explain the code so that someone else can understand the code without stress.
* It makes it erasier to pinpoint a problem when it arises or find the point where the needed change needs to be effected.


- There are basically two types of commenting:
* Single-Line Comment: This uses ' // ' to comment a single line and leave other lines intact.
* Multi-line Comment: This uses ' /* ' and ' */ ' at the beginning and end of the lines of code you want to comment respectively.
