## Go Data Types

This topic introduced me to different types of data:
* int for integers,
* float64 for decimals
* string for strings,
* bool for booleans. 

It helped me understand how Go handles different kinds of values.

E.g;
```go
var number int = 10
var price float64 = 9.99
var name string = "Prina"
var isValid bool = true
```
