## Go Variables

there are 4 types of variables:
- int
- string
- bool
- float64

* int deals with whole numbers.Eg, 19
* string deals with text.eg, "Hello"
* bool deals with true or false statements.
* float64 deals with decimals.Eg, 3.12.

There are two ways to assign variables:
```go
// The first is :
var name string = "excel"
// The second way is :
name := "excel"
```
